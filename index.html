When working with list, table etc. struture from bottom to top:

Step for creating list:

  1. written down the list line by line with 'li' tag
  2. block the 'li' with :
    • un order list ie. 'ul' OR
    • order list ie. 'ol' with para type = a, i, or 1

Step for creating table:

  1. structure out the MAIN ELEMENT 'thead' ; 'tbody' ; 'tfoot' within the'table'.
  2. with CSS style, the main element can be different looking and different control
  3. for each element, layout the number of rows 'tr'. the 'tr' is the main key
  4. for each row 'tr', it either contains table header'th' OR table data 'td'
  5. can use rowspan="2" or colspan="" on the td if necessary
  6. the header 'th' can be either horizontl / verticle
